# 🔧 Tecla Ghost Fix

Um programa simples e eficiente desenvolvido em Python para resolver problemas de **tecla fantasma** em computadores e notebooks.

## 💻 O que é tecla fantasma?
Tecla fantasma é quando uma tecla é acionada sozinha, causando digitação automática, comandos inesperados e dificuldade no uso do computador.

## 🚀 Funcionalidades
- Bloqueio de teclas específicas
- Remapeamento de teclas
- Execução automática em segundo plano
- Interface simples e leve

## 🎯 Objetivo
Permitir que o usuário continue utilizando o computador normalmente, mesmo com teclas defeituosas, sem precisar trocar o teclado.

## 🛠️ Tecnologias utilizadas
- Python
- Biblioteca keyboard
- Tkinter (interface)

## ⚠️ Observação
Este projeto foi criado para fins educacionais e para ajudar usuários com problemas reais de teclado.

## 📦 Download
O arquivo completo está disponível via link externo após aquisição.

## 🔐 Segurança
O programa não coleta dados do usuário e não realiza nenhuma ação maliciosa.
